# 视频的目的

  网路上有很多 VUE 综合应用的视频，但大部份的视频 ，都当你已经了解官网
  所以讲的节奏难免偏快。
  此视频，以官网为重，将官网的观念，以视频搭配代码，实际演示。
  如果官网有看不懂的地方，比对一下，视频跟代码，就会了解了。
  减少你摸索的时间。

# vue.js 2 vue-cli how to start

  sudo npm install -g vue-cli

  vue init webpack my-project
  
  cd my-project
  
  npm install
  
  npm run dev
